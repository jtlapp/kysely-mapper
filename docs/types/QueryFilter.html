Type of the query filter object, which can be passed as an argument to query functions to constrain results. A filter can be any of the following:

  • A key column value, which matches a single key column, if KeyColumns is a tuple with a single element.
  • A key tuple, which matches multiple key columns, if KeyColumns is a tuple with multiple elements.
  • An object, which matches columns against the object's fields. A field value may be an array, in which case the associated column ] matches any of the elements of the array.
  • A raw SQL expression using Kysely's sql template literal tag.
  • An arbitrary Kysely query expression.

A filter is represented as a single value, but the methods that use filters also accept three-argument binary operations.
